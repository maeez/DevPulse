# DevPulse

A GitHub productivity dashboard built with React, TypeScript, Tailwind CSS, React Query, and Zustand.

## Features

- Contribution heatmap, PR cycle time analytics, and live activity feed
- Repository explorer with search, language filter, and sort — state synced to URL
- Stale repo detector, dark/light mode, and skeleton loaders throughout

## Tech Stack

React 18 · TypeScript · Tailwind CSS v4 · Zustand · TanStack React Query · Axios · Recharts · React Router

## Getting Started

```bash
git clone https://github.com/maeez/DevPulse.git
cd devpulse
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) and connect your GitHub Personal Access Token with `repo`, `read:user`, and `read:org` scopes.

> Your token is stored in localStorage only and never sent to any external server.
